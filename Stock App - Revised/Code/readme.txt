To launch the system user will type the following commands in the StockMarketApp 
directory terminal:
NOTE: THESE ARE NECESSARY LIBRARIES FOR RUNNING THE CODE
> pip install flask
> pip install flask_sqlalchemy
> pip install flask_login
> pip install requests

And to run the Code the command below should be ran in the code directory:
> python main.py
and press the link where it says “Running on xxxxxx”, where x represents the link the 
program is hosted.


To run the test:
1. cd to the directory of the test file
NOTE: TO RUN THE TESTS, USER SHOULD USE THE TEST FILE IN THE WEBSITE FOLDER 
INSIDE THE STOCKMARKET CODE FOLDER
2. run: python -m pytest test.py
